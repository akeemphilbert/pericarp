# Configuration Templates

This directory contains configuration file templates for different environments and use cases.

## Purpose

Configuration templates help users set up the library for their specific needs:
- Development environment
- Production environment
- Testing environment
- Different database backends

## Usage

Copy the example configuration file and modify it for your environment:

```bash
cp configs/config.yaml.example config.yaml
```
